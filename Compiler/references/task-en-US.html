Problem Statement

Constants are data elements whose values are known and do not change during program execution.

In the Pascal language, the keyword "Const" is used to describe constants.

The format is: "Const constant_name:type=value;".

Examples:

  1. String constant - the value is represented as a string enclosed in single quotes: "Const str:string='Hello';"

Due to the developed finite state machine grammar G[‹SC›], the syntax analyzer (parser) for string constants will consider the following constant entries correct:

  1. "Const str:string='string';"
  2. "Const he11o:string='Hello, world!';"
  3. "Const he1_1o:string='Hello, world!';"
