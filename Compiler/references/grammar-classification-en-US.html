Grammar Classification

According to Chomsky's classification, grammar G[‹SC›] is automaton.

Rules (1)-(9) belong to the class of right-recursive productions (A → aB | a | ε):

  1. ‹SC› → “Const”‹IdSpace›
  2. ‹IdSpace› → “ “‹Id›
  3. ‹Id› → ‹Letter›{‹Letter›|‹Digit›|“_”}‹TypeAssign›
  4. ‹TypeAssign› → “:”‹Type›
  5. ‹Type› → “string”‹Assign›
  6. ‹Assign› → “=”‹StringBegin›
  7. ‹StringBegin› → “’”‹String›
  8. ‹String› → {‹Letter›|‹Digit›}|‹StringEnd›
  9. ‹StringEnd› → “’”‹EndExpr›

Note that rules must be either all left-recursive or all right-recursive. A combination of both is not allowed.
