Application Functions:

File

  • Create - function to create a file editing window. To save it, use the "Save as" function
  • Open - function opens a file explorer where the user specifies the file to open
  • Save - function to save changes made to the file
  • Save as - function to save the file
  • Exit - function to exit the application. If there are unsaved changes, you will be prompted to save them.

Edit

  • Undo - function to undo changes made
  • Redo - function to redo changes made
  • Cut - function to cut selected text
  • Copy - function to copy selected text
  • Paste - function to paste text from the clipboard into the editor
  • Delete - function to delete selected text
  • Select all - function to select all text in the editor

Reference

  • Call reference - function opens the help HTML document in the browser
  • Program description - function opens the program description HTML document in the browser
