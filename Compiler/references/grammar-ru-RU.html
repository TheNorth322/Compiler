Определение грамматики / Grammar Definition

Определим грамматику строковых констант языка Pascal G[‹SC›] в нотации Хомского с продукциями P:

  1. ‹SC› → “Const”‹IdSpace›
  2. ‹IdSpace› → “ “‹Id›
  3. ‹Id› → ‹Letter›{‹Letter›|‹Digit›|“_”}‹TypeAssign›
  4. ‹TypeAssign› → “:”‹Type›
  5. ‹Type› → “string”‹Assign›
  6. ‹Assign› → “=”‹StringBegin›
  7. ‹StringBegin› → “’”‹String›
  8. ‹String› → {‹Letter›|‹Digit›}|‹StringEnd›
  9. ‹StringEnd› → “’”‹EndExpr›
  10. ‹EndExpr› → “;”

Следуя введенному формальному определению грамматики, представим G[‹SC›] ее составляющими:

  • Z = ‹SC›;
  • VT = {“Const”, “string”, “a”…”z”, “A”…”Z”, “0”...”9”, “:”, “=”, “;”, “_”, “’”, “ “};
  • VN = {‹SC›, ‹IdSpace›, ‹Id›, ‹TypeAssign›, ‹Type›, ‹Assign›, ‹StringBegin›, ‹String›, ‹StringEnd›, ‹EndExpr›, ‹Digit›, ‹Letter›}.
