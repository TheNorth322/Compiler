Функции приложения:

Файл (File)

  • Создать (Create) - функция для создания окна редактирования файла. Для того чтобы сохранить его необходимо воспользоваться функцией "Сохранить как"
  • Открыть (Open) - функция открывает проводник, в котором пользователь указывает файл, который необходимо открыть
  • Сохранить (Save) - функция для сохранения изменений внесенный в файл
  • Сохранить как (Save as) - функция для сохранения файла
  • Выход (Exit) - функция для выхода из приложения. Если есть несохраненные изменения, будет предложено их сохранить.

Правка (Edit)

  • Отменить (Cancel) - функция для отмены внесенных изменений
  • Повторить (Repeat) - функция для возвращения внесенных изменений
  • Вырезать (Cut) - функция для вырезания выделенного текста
  • Копировать (Copy) - функция для копирование выделенного текста
  • Вставить (Paste) - функция вставляет текст из буфера обмена в редактор
  • Удалить (Delete) - функция удаляет выделенный текст
  • Выделить все (Select all) - функция выделяет весь текст в редакторе

Справка (Reference)

  • Вызов справки (Call reference) - функция открывает html-документ справки в браузере
  • О программе (Program description) - функция открывает html-документ описания программы в браузере
